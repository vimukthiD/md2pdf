# Examples

This directory contains example usage of md2pdf.

## Basic Example

```bash
md2pdf document.md
```

## Custom Config Example

```bash
md2pdf --init
# Edit md2pdf.config.json
md2pdf document.md
```

## Theme Example

```bash
md2pdf document.md --theme dark
```

## Custom CSS Example

```bash
md2pdf document.md --css custom.css
```
